# Shortcuts
Super + Enter --> alacritty
Super + E --> pcmanfm-qt
Super + Q --> brave-browser
Super + Shift + Q --> Close focused app 
Super + Shift + R --> Restart i3-wm
Super + M --> Exit i3-wm
Super + R --> rofi
